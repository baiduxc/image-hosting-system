# 专业图床管理系统

## Core Features

- 用户注册登录

- 权限管理

- 本地图片上传

- 网络图片转存

- 图片管理面板

- 链接生成

- 存储统计

- 防盗链绕过

- 多云存储配置

- 管理员系统配置

现代化SPA应用，Vue Router路由管理，用户权限分级，响应式设计，TDesign组件库风格
